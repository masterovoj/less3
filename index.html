Урок №3. Знакомство с CSS.

Подключение CSS файла к html странице

Для подключения CSS файла к html странице необходимо в заголовке (head) прописать тег <link> </link> . Атрибут rel=" " указывает, что мы подключаем CSS документ, атрибут href=" " указывает путь до нашего CSS файла.

<link rel="stylesheet" href="css/style.css">

Типы Селекторов

Селектор тега

При обращении к HTML странице из CSS страницы по селектору тега, свойства, присвоенные этому селектору применятся ко всем одноименным тегам, расположенным на данной странице.

Запись в CSS файле

div{ color: red; }

Запись в HTML

<div class="code">
<p>Lorem ipsum dolor, sit amet consectetur adipisicing elit. Qui, quae.</p>
<p>Далеко-далеко за словесными горами в стране гласных и согласных живут рыбные тексты.</p>
</div>

Результат

Lorem ipsum dolor, sit amet consectetur adipisicing elit. Qui, quae.

Далеко-далеко за словесными горами в стране гласных и согласных живут рыбные тексты.

Селектор класса

Свойства будут применяться к конкретному классу.

Запись в CSS файле

.content{ color: blue; }

Запись в HTML

<div class="block">
Lorem ipsum dolor sit amet, consectetur adipisicing elit. Veniam modi nostrum vel cumque facere quibusdam, voluptate ab a possimus incidunt.
</div>
<div class="content">
Далеко-далеко за словесными горами в стране гласных и согласных живут рыбные тексты. Рукопись эта букв, повстречался своих ему то заманивший океана подпоясал.
</div>

Результат

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Veniam modi nostrum vel cumque facere quibusdam, voluptate ab a possimus incidunt.

Далеко-далеко за словесными горами в стране гласных и согласных живут рыбные тексты. Рукопись эта букв, повстречался своих ему то заманивший океана подпоясал.

Селектор идентификатора

Свойства применятся только элементу с указанным идентификатором.

Запись в CSS

#id-block{ color: green; }

Запись в HTML

<div id="id-block" class="block">
<p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Minus deleniti, itaque maxime tempore quaerat placeat ut aliquid consequatur in harum.</p>
</div>
<div class="block">
<p>Далеко-далеко за словесными горами в стране гласных и согласных живут рыбные тексты. Страна свой эта что ее по всей, пустился вершину раз власти.</p>
</div>

Результат

Lorem ipsum dolor sit amet consectetur adipisicing elit. Minus deleniti, itaque maxime tempore quaerat placeat ut aliquid consequatur in harum.

Далеко-далеко за словесными горами в стране гласных и согласных живут рыбные тексты. Страна свой эта что ее по всей, пустился вершину раз власти.
